###### Bank Management System

<p>It is a console application, where bank employees can login with  credentials and add details of the customer.Withdraw and deposit of the cash can be done as per the customer requirement and Account balance will be reflected accordingly.Employee can view details of all the customers and can search for the required details.</p>
